# G-AI 快速启动指南

## 🚀 一键启动

```bash
./start.sh
```

就这么简单！脚本会自动：
- ✅ 检查依赖
- ✅ 启动后端（端口 8001）
- ✅ 启动前端（端口 8000）
- ✅ 显示服务地址

## 📍 访问地址

启动成功后：

- **前端**: http://127.0.0.1:8000
- **后端 API**: http://127.0.0.1:8001
- **API 文档**: http://127.0.0.1:8001/docs

## 🛑 停止服务

按 `Ctrl+C` 或运行：

```bash
./stop.sh
```

## 📝 其他脚本

- `start_backend.sh` - 仅启动后端
- `start_frontend.sh` - 仅启动前端
- `stop.sh` - 停止所有服务

详细说明请查看 [SCRIPTS.md](SCRIPTS.md)
